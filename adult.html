FAQ

常見問題

我完全不會寫程式，真的能上嗎？

能。課堂只用四個指令（npm install／npm run dev／npm run build／git status），程式全部由 AI 撰寫，你負責用中文把需求講清楚——這正是課程要教的能力。

初階、進階要一起報嗎？

可以只上初階：結束時你已有一個公開上線的網站。進階班需要初階基礎（或已會用 Claude Code 做網頁＋接資料庫），兩堂課用同一個專案，一起上效果最好。

Claude Pro 是什麼？一定要買嗎？

Claude Pro（約 US$20／月）是上課工具的訂閱方案，需自備；行前信會一步步教你註冊，課後可隨時取消。

做出來的網站課後還能用嗎？

能。網站、資料庫、程式碼全部在你自己的帳號裡，課後直接繼續用；進階班更會教你維運與回復，出問題救得回來。

上完有證書嗎？

課程教學法採 Anthropic 官方「AI Fluency 4D 框架」。課後我們會在群組帶你挑戰官方免費線上課（AI Fluency: Framework & Foundations），通過測驗即獲 Anthropic 官方完課證書——課堂就是它的實戰版。

下課後卡關怎麼辦？

專屬課程群組就是為這個存在的——老師陪伴答疑、同學互相支援，作品可以一直迭代下去。
