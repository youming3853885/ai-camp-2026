AI 建站一日營

6 小時，帶一個公開網址回家

你現在看的這個網站，就是用今天要教的方法做的。

今日議程

  • 9:00–9:30 震撼開場 ▸ 老師現場用 Claude Code 五分鐘生一個網站。
    你的任務

    先連上教室 Wi-Fi，打開兩個分頁：Gemini（gemini.google.com）和這個網站。看老師表演就好。

  • 9:30–10:30 AI 素養 ＋ 提示詞實戰 ▸ 角色／脈絡／格式三要素、迭代追問、辨識幻覺。
    你的任務

    打開講義第 1 篇，複製「AI 訪談你」的 prompt 貼給 Gemini，跟它聊出你的自介 —— 下午網頁要用。

  • 10:40–12:00 vibe coding 初體驗 ▸ Gemini Canvas 生你的第一個網頁。
    你的任務

    用講義的起手 prompt 讓 Canvas 生一頁網站，然後用「說的」改三個地方：標題、主色、一張卡片。

  • 12:00–13:00 午休 · 吃飽再戰
  • 13:00–14:10 主專案：學習歷程網頁 ▸ 把模板變成你自己的作品。
    你的任務

    打開模板骨架（在講義裡，整份複製），把裡面所有【請填入】換成你的內容——共五個區塊（用上午的自介 ＋ 截圖），丟給 Canvas 迭代到你滿意。

  • 14:10–15:00 部署上線 ▸ GitHub Pages 拿到你的公開網址。
    你的任務

    照講義第 2 篇：GitHub 建 repo → 上傳 index.html → 開 Pages → 拿到你的公開網址。舉手讓老師看你的網址！

  • 15:00–15:35 NotebookLM 打包 PDF ▸ 把今天的東西變成一份學習報告。
    你的任務

    把講義 ＋ 你的筆記餵給 NotebookLM，生成學習報告，加上反思和你的網站 QR code，匯出 PDF。

  • 15:35–16:00 發表 ＋ 認證 ▸ 唸出你的網址，領走今天的成果。
    你的任務

    輪流唸出你的網址 ＋ 一句今天最有感的事。想拿認證的看下方認證說明。
